# おわりに

<!-- textlint-disable ja-technical-writing/ja-no-weak-phrase -->

本書では、DevContainerについて、筆者の意見も含めて解説しました。

筆者自身もこの本でまとめたことで、様々あるDevContainerの構築方法から何を選ぶべきか判断できるようになりました。
最近はローカル開発と同じようにDevContainerが使えるようになってきました。
筆者としてはこのDevContainerで作った環境を使って、LLMコーディングエージェントを活用した開発を進めたいと考えています。

本書を通して読者の理解が深まり、DevContainerの活用や、LLMコーディングエージェントの安全な運用に役立てば幸いです。
