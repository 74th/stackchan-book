## スタックチャンとは

スタックチャンとは、M5Stack清貧を使って、顔の付いたカワイイロボットを構築するコミュニティ主導のムーブメントです。

ししかわ（[@meganetaaan](https://x.com/meganetaaan)）さんという方が製作され、オープンソースハードウェアとして公開されました。
コミュニティの誰もが、M5Stack製品を使って製作して、お家に迎えることができます。

> TODO: ししかわさんの Protopedia のスタックチャンの記事のURL

> スタックチャン（Stack-chan） | ProtoPedia<br/>https://protopedia.net/prototype/2345

つぶらな瞳に、ふわっとした動作がかわいく、魅了された方も多かったのではないかと思います。

ファームウェアには、robo8080（[@robo8080](https://x.com/robo8080)）さんがChatGPTと接続してAIによる応答が可能になったAIスタックチャンが登場しました。
音声合成には、無料で利用できるようにしていただいているVoicevoxが用いられています。
ChatGPTのAPI課金で手軽にAIチャットボットが構築できるようになり、この魅力により多くの方が製作にトライされたと思います。

だれでもスタックチャンの製作ができるように、タカヲ（[@mongonta555](https://x.com/mongonta555)）さんが個人でスタックチャン筐体の3Dプリント作品を販売されています。
こちらには、akita11さんのボードと組み合わせて利用でき、実装済みボードや小型ブレッドボードを用いることで、はんだ実装なしで製作にトライすることができます。
これにより、電子工作の知識や道具がない方でも、スタックチャンを迎えることができたと思います。

現在は、ししかわさんの作成されたスタックチャン以外にも、スタックチャンと表現した複数の作品が制作、公開されています。

ししかわさんの原典のスタックチャンは、以下のリポジトリで公開されています。

> stack-chan/stack-chan: A JavaScript-driven M5Stack-embedded super-kawaii robot.<br/>https://github.com/stack-chan/stack-chan

## スタックチャンの主な部品

スタックチャンには様々な製作がありますが、ここではオーソドックスな組み合わせの部品構成を説明します。

おもに以下の部品からできています。

1. M5Stackコアシリーズ
2. サーボモータ
3. 接続基板
4. ケース
5. 充電池
6. ファームウェア

M5Stackコアシリーズ（以下コア）は、IoT開発キットとして、WiFi接続可能なマイコンに、ディスプレイ、バッテリー、マイクなどのデバイスが一体になった製品です。
ここに、動きを加えるサーボモータを接続します。
骨格となるケースに、サーボモータとコアを組み込んで、スタックチャンの形になります。
サーボモータとコアを接続するのが接続基板です。
主にケースに合わせて基板が存在している形です。

充電池はなくても良いですが、充電池がないとUSBでの給電が不安定になることが多く、充電池も用意した方が良いです。

本節では、それぞれの部品について触れた後に、可能な組み合わせについて説明します。

部品とは異なりますが、頭脳となるファームウェアが必要です。

### M5Stack製品

M5StackコアシリーズとよばれるIoT開発キットが必要です。

スタックチャンの製作には、主にM5Stack製品のBASIC、M5Stack Core2、CoreS3が使われます。
機能としては以下のような違いがあります。

| 名前         | BASIC  | Core2  | CoreS3/Lite | CoreS3 SE |
| ------------ | ------ | ------ | ----------- | --------- |
| MCU          | ESP32  | ESP32  | ESP32-S3    | ESP32-S3  |
| ディスプレイ | あり　 | あり   | あり        | あり      |
| タッチセンサ | -      | あり   | あり        | あり      |
| マイク       | -      | あり(※) | あり        | あり      |
| カメラ       | -      | -      | -         | あり      |

CoreS3は利用するMCUがESP32-S3と、高性能な後継品になっています。
この後本書で利用するWake-up Word DetectionライブラリESP-SRは、利用するライブラリArduinoではESP32-S3用にしか提供されていません。
よって、ESP32-S3を搭載した、CoreS3、CoreS3 Lite、CoreS3 SEのどれかを利用してください。
なお、M5Stack公式スタックチャンキットでは、CoreS3が搭載されています。

なお、本書の製作するスタックチャンファームウェア以外であれば、ESP32を持ちいたBASIC、Core2で動作するものもあります。
Core2のマイクはMBus（積み重ねる形で利用するモジュール用の接続ソケット）に、蓋をする基板に付属しています。

## 自分で製造する方法

タカヲさん版のケースを用いる場合の、自分で製造する方法を簡単に説明します。

ここからの内容は、KiCadのバージョンアップや、3Dプリントサービスの都合で失敗する可能性があります。
自己責任で進めてください。

PCBの製造と、ケースの3Dプリントは、JLCPCBという中国の製造サービスで行うことができます。
筆者の場合は、PCBの製造は発注し、ケースの3Dプリントは自宅の3Dプリンタ（Anker Make M5C）で行いました。

### 基板の発注方法

SG90用の基板データのリポジトリをクローンします。

> https://github.com/akita11/Stack-chan_Takao_Base

初めてKiCadを利用する方はインストールをしてください。
KiCadの起動後に「プラグイン&コンテンツマネージャ」を開き、「Fabrication Toolkit」をインストールしてください。
このプラグインの説明に「JLCPCB」の単語があるものをインストールしてください。
「プラグイン&コンテンツマネージャ」では、「インストール」ボタンを押した後に、「保留中の変更を適用」ボタンを押して、インストールを適用する必要があります。
拡張子.kicad_projファイルがプロジェクトファイルです。
このファイルをKiCadで開き、「PCBエディタ」を開きます。
メニューの「ツール」から「外部プラグイン」→「Fabication Toolkit」を開き「Generate」を押します。
すると、JLCPCBの製造用ZIPファイルが生成されます。

次にJLCPCBのサイトでアカウントを作成し、ログインしてください。

> JLCPCB<br/>https://jlcpcb.com/jp/

ログイン後、「発注する」ボタンを押して、「製品を選択」で「PCB」を選択してください。
その後、「ガーバーファイルを追加」のところで、先ほどKiCadのプラグインで生成したZIPファイルをアップロードしてください。
オプション等は初期設定で変更する必要はないでしょう。
PCBカラーを変更したり（なお、組み立てると見えません）、表面仕上げで「HASL=鉛入りはんだ」から「無鉛HASL」に変更しても良いでしょう。

3Dプリントも同時に注文できますが、PCBノミ注文する場合にはそのまま、注文に進めてください。
この時、発送方法に「OCS NEP」、「OCS EXPRESS」を選択すると、かなり安く発送できます。

### 3Dプリントの発注

3Dプリントの発注はJLCPCBでもできます。

スタックチャンのリポジトリをクローンします。

> https://github.com/stack-chan/stack-chan

`case/contributed/mongonta_case_for_SG90_and_M5GoBottomBoard/case_for_SG90andM5GoBottomBoard`のディレクトリにある、以下のSTLファイルを利用します。

- bracket_SG90_v2.0.stl
- shell_SG90_Goplus2_module_v1.0.stl
- feet_SG90_v1.0.stl

JLCPCBにログイン後、「発注する」ボタンを押して、「製品を選択」で「3Dプリント」を選択してください。
その後、「3Dファイルの追加」のところで、STLファイルをアップロードしてください。

3Dプリントの製造方法（項目「3D技術」）について、「FDM（Plastic）」を選択できれば良いのですが、製造する部品にFeetがサイズに満たないため、「SLA（Resin）」を選択する必要があります。
素材は「9600 Resin」を選択すると良いでしょう。
筆者は他の素材については調べていません。
「商品概要」の項目を埋める必要があり、「DIY/Hobby」→「その他」→「Robot Parts」と選択しています。
それぞれの部品を発注しましょう。

発送はPCBと3Dプリントでまとめられます。
